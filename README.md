## Inventory Management System


This system will be used to register stock deliveries, maintain current stock and register shipments.

## Steps to run application

## Api setup

```
cd api

npm install --legacy-peer-deps
```

## Run Api

```
npm run dev
```

## Project setup

```
cd my-app

npm install --legacy-peer-deps
```
